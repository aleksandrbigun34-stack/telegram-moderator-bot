# Telegram Bot для модерації чатів 🛡️

Професійний інструмент для автоматичної модерації Telegram-чатів, написаний на мові Python з використанням бібліотеки pyTelegramBotAPI.

## 🚀 Основний функціонал:
*   Фільтрація спаму та нецензурної лексики: Автоматично аналізує повідомлення та видаляє заборонені слова.
*   Система попереджень (Warns): Веде облік порушень для кожного користувача за допомогою локальної бази даних JSON.
*   Автоматичне блокування: При накопиченні критичної кількості попереджень (3 варни) бот автоматично блокує порушника в чаті.
*   Інтерактивні кнопки: Зручне Inline-меню для ознайомлення з правилами чату.
*   Адмін-команди: Можливість ручного керування попередженнями (/unwarn тощо).

## 📦 Як запустити проект:
1. Встановіть бібліотеку: pip install pyTelegramBotAPI
2. Вставте токен бота в main.py
3. Запустіть: python main.py
